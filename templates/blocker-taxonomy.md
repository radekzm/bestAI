# Blocker Taxonomy

| Group | Canonical blockers | Typical escalation |
|------|---------------------|--------------------|
| DATA | STALE_DATA, NEED_DATA | Request data source / run measurement |
| TECH | BUILD_ERROR, DEPLOY_FAIL, API_REJECT | Change approach, isolate failure, rerun tests |
| PRODUCT | UX_UNCLEAR, VALUE_UNPROVEN, SCOPE_CREEP | Clarify scope and success criteria |
| EXTERNAL | BLOCKED_EXTERNAL | Ask user for external action/access |
| BUSINESS | PRICING_UNVALIDATED, MARKET_FIT_UNKNOWN | Run validation experiment |
